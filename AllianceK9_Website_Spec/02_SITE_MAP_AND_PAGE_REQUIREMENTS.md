# Alliance K9 Site Map and Page Requirements

## MVP Routes

- `/` — Homepage
- `/parents` — Ranger × Mila / Meet the Parents
- `/puppies` — Current litter / available puppies
- `/puppies/[slug]` — Reusable puppy profile template
- `/training` — Training overview, process, results, FAQ
- `/apply` — Puppy application / reserve page
- `/gallery` — Media gallery
- `/contact` — Contact and inquiry routing

## Page Requirements

### Homepage
Purpose: establish brand, emotion, trust, and direct users toward puppies, application, training, or contact.

Required sections:
- Cinematic hero using the Mila + puppy hero image when available
- Primary CTA: View Puppies
- Secondary CTA: Apply / Inquire
- Trust strip
- Ranger × Mila preview
- Puppy preview grid
- Training preview
- Final CTA

### Parents Page
Purpose: prove the foundation of the litter.

Required sections:
- Ranger profile
- Mila profile
- Pairing rationale
- Working bloodline / temperament discussion
- Parent gallery
- CTA to puppy listings

### Puppies Page
Purpose: list the litter and move visitors to individual puppy profiles.

Required sections:
- Litter overview
- Birth date: March 21, 2026
- Litter size: 12 puppies, 6 males and 6 females
- Puppy grid with availability badges
- SOLD/Reserved puppies remain visible
- Application CTA

### Puppy Profile Template
Purpose: sell and explain each individual puppy.

Required sections:
- Puppy hero image
- Name and status badge
- Sex, collar, birthdate, weight notes, traits, best fit
- Full description
- Puppy-specific gallery
- Early development checklist
- Ranger × Mila cross-link
- Similar puppy recommendations
- Final CTA

### Training Page
Purpose: create secondary revenue path and reinforce breeder credibility.

Required sections:
- Hero
- Training service cards
- Training philosophy
- Process timeline
- Results/testimonials section
- FAQ
- Training inquiry CTA

### Application Page
Purpose: qualify buyers and guide them into the placement process.

Required sections:
- Placement philosophy
- Process timeline
- Application form
- Deposit explanation
- FAQ / reassurance
- Final CTA

### Gallery Page
Purpose: showcase the dogs, pups, training, and program visually.

Required sections:
- Masonry gallery
- Filters if practical
- Lightbox
- Portrait-friendly layout
- Lazy loading

### Contact Page
Purpose: route general, puppy, and training inquiries.

Required sections:
- Contact form
- Puppy inquiry CTA
- Training inquiry CTA
- Contact/social details
- Response expectation copy
