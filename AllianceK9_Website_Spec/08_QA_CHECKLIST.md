# Alliance K9 QA Checklist

## Pre-Launch Functional QA

- [ ] Homepage loads.
- [ ] Navigation works on desktop.
- [ ] Mobile menu works.
- [ ] Puppies page loads.
- [ ] Every puppy card links to correct profile.
- [ ] Puppy profile pages render from content.
- [ ] SOLD / Reserved / Available badges are accurate.
- [ ] Application form submits.
- [ ] Contact form submits.
- [ ] Training inquiry form submits.
- [ ] Gallery loads.
- [ ] Lightbox works if implemented.
- [ ] Footer links work.

## Visual QA

- [ ] Dark luxury aesthetic matches approved mockups.
- [ ] Gold accents are restrained.
- [ ] Typography uses approved hierarchy.
- [ ] No page feels like a generic breeder template.
- [ ] Mobile layouts are clean.
- [ ] Puppy photos are not awkwardly cropped.
- [ ] Eyes, ears, muzzles, and collars are preserved where important.
- [ ] Portrait images look intentional.

## Content QA

- [ ] No lorem ipsum remains.
- [ ] Puppy status is current.
- [ ] Puppy descriptions are accurate.
- [ ] Birthdate and litter details are correct.
- [ ] Deposit/payment language is accurate.
- [ ] Contact information is correct.

## Performance QA

- [ ] Images are optimized.
- [ ] Lazy loading used for gallery images.
- [ ] No massive original photos are served directly.
- [ ] Site loads acceptably on mobile.

## SEO / Sharing QA

- [ ] Every page has a title.
- [ ] Every page has a meta description.
- [ ] Puppy profile pages have unique metadata.
- [ ] Social share images are configured where practical.
- [ ] Image alt text is present.

## Launch Gate

The site may launch when these are complete:

- [ ] Homepage
- [ ] Puppies page
- [ ] Individual puppy profiles
- [ ] Apply page
- [ ] Contact page
- [ ] Working forms
- [ ] Accurate puppy statuses
- [ ] At least one strong image per available puppy
