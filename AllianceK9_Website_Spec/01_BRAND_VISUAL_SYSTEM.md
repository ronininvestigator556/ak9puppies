# Alliance K9 Brand & Visual System

## Approved Direction

**A2 — Heritage Luxury**

Alliance K9 should feel premium, cinematic, calm, and credible. The brand should communicate working-dog seriousness without looking militarized, gimmicky, or overly tactical.

## Positioning Statement

**Working-Bred. Family-Raised. Purpose-Matched.**

## Brand Personality

Alliance K9 should present as:

- Professional
- Refined
- Confident
- Warm
- Knowledgeable
- Purpose-driven
- Trustworthy
- Working-dog credible

Alliance K9 should not present as:

- Tactical cosplay
- Pet-store cute
- Classified-ad cheap
- High-pressure sales
- Generic breeder template
- Overly corporate or sterile

## Color Palette

| Role | Name | Hex | Usage |
|---|---|---:|---|
| Primary Background | Midnight Black | `#0D0D0D` | Main page backgrounds, hero sections |
| Secondary Background | Graphite | `#1B1B1B` | Cards, panels, form blocks |
| Accent | Burnished Gold | `#B08D57` | Buttons, borders, icons, section accents |
| Main Text | Ivory | `#F2F2F2` | Primary copy and headings |
| Secondary Text | Stone Gray | `#A3A3A3` | Subtext, metadata, helper text |

### Color Rules

- Gold should be restrained and premium, not flashy.
- Gold should function as trim, not decoration overload.
- Avoid bright yellow, neon green, tactical green, camo palettes, or aggressive red except where necessary for status indicators.
- SOLD / Reserved badges should be tasteful and consistent with the luxury aesthetic.

## Typography

### Headings

Recommended: `Cinzel`

Purpose:

- Heritage
- Elegance
- Pedigree feel
- Premium editorial tone

Fallback:

```css
font-family: "Cinzel", Georgia, serif;
```

### Body

Recommended: `Inter`

Purpose:

- Clean readability
- Modern UI clarity
- Strong mobile performance

Fallback:

```css
font-family: "Inter", system-ui, sans-serif;
```

## Layout Principles

Use:

- Large cinematic hero sections
- Strong photography
- Dark overlays for legibility
- Generous spacing
- Clear calls to action
- Editorial page rhythm
- Premium card layouts
- Subtle motion

Avoid:

- Dense clutter
- Heavy ornamentation
- Busy backgrounds behind text
- Excessive animations
- Forced square image crops
- Clipart-style dog graphics

## Button Styles

### Primary Button

Use for main actions:

- View Puppies
- Apply / Inquire
- Contact Alliance K9

Style:

- Burnished gold background
- Dark text or ivory text depending contrast
- Slight radius
- Premium hover transition

### Secondary Button

Use for supporting actions:

- View Parents
- Learn More
- View Gallery

Style:

- Transparent or graphite background
- Gold border
- Ivory text
- Gold hover accent

## Card Styles

Cards should use:

- Graphite background
- Thin gold or muted border
- Soft shadows
- Rounded corners
- Clear hierarchy
- Portrait-friendly image areas

Puppy cards should display:

- Puppy photo
- Name
- Sex
- Collar color
- Status badge
- Trait chips
- Short description
- CTA

## Status Badges

Allowed statuses:

- Available
- Pending
- Reserved
- Sold
- Evaluation Hold

Rules:

- Status must be accurate.
- SOLD and Reserved puppies may remain visible as social proof.
- Do not hide sold puppies automatically.
- Use subtle social proof language.
- Avoid high-pressure urgency language.

Example copy:

> Iroh has found his home. Looking for a similar prospect? View Zuko, Bumi, or Hakoda.

## Voice and Copy Style

The site should use confident, professional copy.

Use language like:

- purpose-bred
- professionally evaluated
- working potential
- active family companion
- temperament matching
- handler engagement
- thoughtful placement

Avoid overpromising:

- guaranteed protection dog
- perfect family dog
- elite police dog certainty
- fearless forever
- will protect your family without training

Use cautious evaluation language:

> Early observations suggest...

> Shows promising signs of...

> Best suited for...

> Based on current temperament and drive observations...

## Photography Direction

Preferred aesthetic:

- Real Alliance K9 photos
- Cinematic natural light
- Warm but controlled grading
- Rich blacks
- Natural coat color
- Emotional expression
- Working-dog authenticity

Photo categories:

- Hero-grade portraits
- Parent profiles
- Puppy profile images
- Group litter images
- Training/process images
- Social proof/family placement images
- Development timeline images

## Motion and Interaction

Use subtle UI motion only:

- Card hover lift
- Gold border glow/fade
- Soft image scale on hover
- Smooth accordion expansion
- Gentle page transitions if easy

Avoid:

- Bouncing buttons
- Flashing effects
- Excessive parallax
- Distracting animations

## Accessibility Requirements

- Maintain strong color contrast.
- Buttons must be keyboard-focusable.
- Forms must include labels.
- Images need meaningful alt text.
- Navigation must be usable on mobile.
- Do not convey puppy status by color alone; include text labels.

## Design North Star

The final site should feel like:

> A premium European-style working-dog program with modern professionalism, emotional warmth, and clear placement integrity.
